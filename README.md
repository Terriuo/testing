## üöÄ Getting Started
`
### Prerequisites

- [Node.js & npm](https://nodejs.org/)
- [Yarn](https://yarnpkg.com/) (optional)
- [MetaMask](https://metamask.io/) or compatible crypto wallet
- [Truffle](https://www.trufflesuite.com/) / [Hardhat](https://hardhat.org/) (for smart contract development)

### Installation

```sh
git clone https://github.com/OneCrispyDuck/web3-final-year-project.git
cd web3-final-year-project
npm install
# or
yarn install
```

### Running the App

```sh
npm start
# or
yarn start
```

### Deploying Smart Contracts

```sh
# Example with Hardhat
npx hardhat compile
npx hardhat test
npx hardhat run scripts/deploy.js --network [network]
```
